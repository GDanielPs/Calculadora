# 📱 Calculadora Android

Aplicación móvil de **calculadora** desarrollada en **Android Studio** utilizando **Java y XML**.

Esta aplicación permite realizar operaciones matemáticas básicas y fue creada como parte de una actividad académica del módulo de **programación de aplicaciones móviles**.

---

## 🧮 Funcionalidades

La aplicación incluye las siguientes operaciones:

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- 📊 Porcentaje
- 🔄 Cambio de signo (±)
- 🔢 Números decimales
- 🧹 Botón AC para limpiar la pantalla

---

## 🎨 Interfaz

El diseño de la aplicación está inspirado en la **calculadora de iPhone**, utilizando:

- Fondo negro
- Botones de números en gris oscuro
- Botones de funciones en gris claro
- Botones de operaciones en color naranja

La interfaz fue desarrollada utilizando **XML** con los siguientes componentes:

- `LinearLayout`
- `GridLayout`
- `TextView`
- `Button`
